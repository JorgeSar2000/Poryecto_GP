AmbarReport4Society

FixMyStreet es un proyecto de código abierto destinado a la administración de sitios web los cuales tienen como finalidad alertar sobre problemas existentes dentro de una zona determinada para luego ser alertado a las autoridades pertinentes.

OBJETIVOS GENERALES

  1. Implementar una herramienta cívica de monitoreo y geolocalización que permita a la sociedad reportar incidencias desde diferentes puntos del país a las entidades correspondientes.
  2. Disminuir la brecha que existe entre el tiempo en el que ocurre un inconveniente y el tiempo en el cual acogen la petición de ayuda por parte de los ususarios.
  3. Agilizar el proceso de información de manera rápida y precisa mediante un aplicativo móvil con ayuda de la geolocalización.

OBJETIVOS ESPECÍFICOS

  1. Localizar de manera inmediata los distintos inconvenientes dentro de una comunidad.
  2. Realizar procesos eficaces, capaces de solventar dentro de un aplicativo móvil percances de una zona determinada.
  3. Buscar rápida y precisamente aquellas autoridades que están capacitadas para resolver problemas dados por el usuario en tiempo real.

INTRODUCCIÓN

Durante los últimos años se han presentado problemas referentes al sector público como el estado de las calles, poca iluminación en determinados sectores de la cuidad, problemas con el alcantarillado, etc., consecuencia de ello se producen altercados entre autoridades competentes quienes por falta de tiempo no le dan la respectiva importancia al caso y la población quienes sustentan sus reclamos bajo sus respectivos derechos. Para ello fue creado FixMyStreet, cuya metodología es intuitiva con el usuario ya que para reportar un problema lo único que necesita es pegar la ubicación en donde está presente el problema y su ping dentro de un mapa para luego informarlo a las autoridades del caso. La forma en la cual se conecta FixMyStreet con las diferentes autoridades es mediante un correo electrónico (dependiendo de la zona en donde se encuentre la avería, esta será transmitida a la autoridad más cercana) o mediante Open311 la cual tiene como propósito u objetivo generar el un ambiente óptimo para socialización de asuntos públicos.

También cuentan con un apartado móvil en donde es imprescindible que FixMyStreet esté funcionando correctamente ya que el teléfono envía la información a través de la plataforma. La aplicación utiliza PhoneGap (Framework utilizado para la creación de aplicaciones móviles).

FixMyStreet es una aplicación web escrita en Perl, la cual contiene múltiples métodos de instalación para PC ya sea con el uso de Docker (tecnología de creación de contenedores que permite la creación y el uso de contenedores de Linux), uso de un script para la instalación en servidores Debian o Ubuntu o ya sea una instalación manual por parte del usuario.

CORRIENDO FIXMYSTREET EN MÓVIL

El proyecto utiliza Apache Cordova para la producción de aplicaciones tanto en Android como en iOS.
Su estructura básica se conforma de JavaScript, HTML, CSS y demás archivos de imagen. Templates (plantillas con cadena para traducir). Locale (archivos de traducción gettext) y guiones de ayuda para la traducción.
Su estructura más compleja se basa en archivos CSS, archivos de proyecto JavaScript, archivos de vista de red troncal y bibliotecas JavaScript.
La aplicación móvil además de presentar una interfaz intuitiva permite personalizar la misma por medio de hojas de estilo propias.

La utilización de gettext es importante para la traducción, la misma que se apoya de una serie de archivos con plantilla que aplican el módulo Template Tookit Perl.
Los scripts se basan en los utilizados para el sitio web de FixMyStreet.

JUSTIFICACIÓN DEL PROYECTO

Al hablar de problemas públicos dentro de nuestra cuidad encontramos gran cantidad de despreocupación por mantener nuestra cuidad en buenas condiciones. Los cuidadanos reportan día a día diferentes reclamos/peticiones para el reconstrucción de algunas zonas afectadas por los distintos factores ya sean ambientales o humanos. El problemas principal radica en que las autoridades quienes no toman cartas rápidamente en el asunto y genera mal estar entre los cuidadanos.
FixMyStreet cumple con la función de aminorar el tiempo en los cuales se reportan tanto los problemas emitidos por los cuidadanos en cuanto a sector público se refiere como un comunicado por parte de alguna autoridad que se vaya a encargar del caso. Su utilización es bastante sencilla ya que sólo necesitas de dos pasos para reportar algún inconveniente presentado.
