PROCESO DE INSTALACIÓN EN PC

Instalación

Para la instalación de la plataforma FixMyStreet, utilizaremos un entorno virtual que nos permitira recrear un ambiente bajo un sistema operativo en común. Nos vamos a regir o instruir en dos herramientaas para la instalacion, las cuales serán Docker y VirtualBox.

Introducción a las herramientas

Para la instalación de FixMyStreet haremos uso de dos herramientas que trabajan como entornos externos o "invitados" dentro de nuestros ordenadores que nos permiten emular un ambiente igual al de nuestro PC pero con especificaciones dadas desde el operador principal como pueden ser estas el sistema operativo, cantidad de RAM utilizada por una nueva máquina virtual o bien sea la cantidad de núcleos usada por la misma.

VirtualBox

¿Qué es VirtualBox?

Oracle VM VirtualBox, más conocida como VirtualBox es un software de virtualización desarrollado por Oracle Corporation. Mediante esta herramienta es posible instalar sistemas operativos adicionales conocidos como "sistemas invitados", dentro de un sistema operativo "anfitrión".
Este es uno de los software que utilizaremos para la aplicación de FixMyStreet.
NOTA: ANTES DE PROCEDER A CREAR UNA NUEVA MÁQUINA VIRTUAL EN WINDOWS, ES NECESARIO DESCARGAR UNA IMÁGEN ISO DEL SISTEMA OPERATIVO WINDOWS10 x64bits.
ADEMÁS DE ELLO DEBERÁ TENER ACTIVADO LA OPCIÓN DE VIRTUALIZACIÓN, SI ESTA ÚLTIMA NO SE ENCUENTRA ACTIVADA, DEBERÁ SER ACTUALIZADA DESDE LA BIOS

Proceso de Instalación paso a paso

  1. Debemos descargar el software desde su página oficial.
  2. En este paso debemos en primer lugar añadirle un nombre a la máquina virtual, luego elegir el sistema operativo (en este caso de 64 bits).
  3. Como es una máquina virtual nueva debemos elegir que él mismo cree un disco duro virtual.
  4. Especificamos la ubicación del archivo.
  5. Al momento de asignarle una cantidad de disco duro podemos sasignar cualquier valor ya que VirtualBox genera un espacio dinámico dentro del disco duro físico.
  6. Escogemos la opción de "Reservado dinámicamente" y en cuanto las opciones de los archivos del disco duro podemos elegit entre las primeras tres que son nativas tanto de VirtualBox, VMware y de Windows respectivamente.
  7. Al momento de ya haber creado nuestra máquina virtual, procedemos a montar la ISO de nuestro sistema operativo Windows10 x64bits dando click derecho sobre la máquina creada y entrando al apartado "Configuración".
  8. Dentro de este apartado en la opción "Sistema" podemos elegir la cantidad de núcleos que deseamos que utilice la máquina.
  9. Dentro del apartado "Almacenamiento" debemos montar nuestra ISO del sistema operativo antes descargado, para ello vamos al disco "Vacío" y dentro de este iremos al apartado del disco donde entraremos al explorador de archivos y buscaremos la carpeta donde se encuentre nuestra imagen ISO para montarla.
  10. Realizado estos pasos click en "Aceptar" y nuestra máquina virtual estará creada.
